1. This project is built using React.Js and is bootstrapped using create-react-app.
2. To run the app:
   a. Clone the app into your local system.
   b. Open the cloned app's folder in any code editor/ide. (preferably VS Code)
   c. Open up a new terminal.
   d. Type "npm install" without the double quotes in the terminal and hit enter to install all the dependencies required for the project.
   e. Type "npm start" without the double quotes to run the application in the browser. The app runs on localhost port 3000.
   f. In case the app doesn't start and there's an error, make sure no other application is using the port 3000.
